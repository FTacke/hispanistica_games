 # Refactoring Baseline (v2)

 **Stand:** 29. Januar 2026  
 **Zweck:** Kompakter Referenzstand für die aktuelle v2-Mechanik ohne Legacy-Altlasten.

 ---

 ## Mechanik – Kurzüberblick

 - **Levels:** 3 (4 / 4 / 2 Fragen)
 - **Difficulty:** 1–3
 - **Timer:** 40s (named) / 240s (anonym) + 10s Media-Bonus
 - **Joker:** 2× pro Run (50:50)
 - **Scoring:** Punkte pro Frage + Level-Bonus bei perfektem Level

 ---

 ## Content-Regeln (v2)

 - `difficulty` nur 1–3
 - Pro Run: 4 Fragen D1, 4 Fragen D2, 2 Fragen D3
 - Prompts/Answers/Explanations unterstützen **bold** und *italic* (kein HTML)

 ---

 ## API-Kurzfakten

 - Run = 10 Fragen
 - Level-Ende: Index 3, 7, 9
 - Leaderboard: `total_score DESC`, `created_at ASC`

 ---

 **Hinweis:** Diese Baseline ist bewusst v2-only und enthält keine Legacy-Details.
